# Images Directory

This folder is for storing all images related to Chaitanya Resort website.

## Recommended Images

### Hero/Header Images
- `beach-hero.jpg` - Main hero image for homepage (1920x1080px recommended)
- `resort-exterior.jpg` - Resort exterior for About page
- `restaurant-interior.jpg` - Restaurant interior for Restaurant page

### Room Images
- `ocean-view-suite.jpg` - Ocean view suite room
- `beachfront-room.jpg` - Deluxe beachfront room
- `garden-view-room.jpg` - Garden view room
- `family-suite.jpg` - Family suite

### Team Images
- `team-member-1.jpg` - Management team photo
- `team-member-2.jpg` - Hospitality staff photo
- `team-member-3.jpg` - Culinary team photo

### General Images
- `resort-feature-1.jpg` - Beachfront location
- `resort-feature-2.jpg` - Pool area
- `resort-feature-3.jpg` - Dining area
- `resort-feature-4.jpg` - Sunset view

## Image Guidelines
- Use high-quality images (minimum 1920px width for hero images)
- Optimize images for web (JPEG format recommended)
- Maintain aspect ratios for consistency
- Use descriptive filenames

## Note
Currently, the website uses CSS gradient placeholders. Replace the background-image URLs in `css/styles.css` with actual image paths once images are added.
